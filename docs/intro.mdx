# Onboard your Game

Welcome developers to Luxodd! We're thrilled you're considering bringing your creative vision to our unique arcade platform. Our goal is to forge a strong partnership, offering players engaging experiences centered around skill and our innovative "strategic betting" system ‚Äì where players wager on their own abilities, not just chance. This guide outlines the journey your game will take from your development environment to being playable on Luxodd arcade cabinets across various locations.

## Your Game's Path to Production

Getting your game onto the Luxodd platform involves a clear, structured process designed for quality and smooth integration. Below, we detail the steps involved, from initial signup to seeing your game live on merchant arcades.

<Step 
  number="1" 
  title="Developer Account Registration & Verification" 
  imagePath="/img/developer-account-registration-infographic.png"
>
  :::note
  Below steps can be skipped if you already have access to Luxodd Game Dev Portal.
  :::

  1. Fill out our [form](https://forms.gle/ixqvtC9uKjJUNL5k8)
  2. We will verify your account.
  3. If accepted you will receive an email with your login information to the <AdminLink target="_blank">Luxodd Admin Portal</AdminLink>
     - You'll also be issued an account (if not already made) to our <AppLink target="_blank">Luxodd Game Server</AppLink>
</Step>

<Step 
  number="2" 
  title="Register Your Game (Draft State)" 
  imagePath="/img/step-2-register-your-game.png"
>
  1. Log in to our <AdminLink target="_blank">Luxodd Admin Portal</AdminLink>
  2. Navigate to ***Games > New***
  3. Fill in the following minimum details:
     - Game Title
     - Game Description
  4. Click ***Save***. The game will be saved in the _Draft_ state.
</Step>


<Step 
  number="3" 
  title="Dev API Token & Integration (Draft State)" 
  imagePath="/img/dev-api-token-infographic.png"
>
  üì¶ **Download Plugin & Documentation**
    - Download the [Luxodd Arcade Unity Plugin](https://github.com/luxodd/unity-plugin/releases)
    - view unity Plugin <UnityDocsLink target="_blank">Documentation</UnityDocsLink>

  üîå **Integration Summary**
    - In The Admin Portal, generate your ***Developer Token***
    - Import the Plugin into your Unity project
    - Use the token to authenticate game API calls
    - Refer to the [Arcade Launch Overview](/docs/arcade-launch/unity-plugin/overview) for detailed setup instructions

    :::info
    _üí° Use the <AppLink target="_blank">app.luxodd.com</AppLink> API for testing during development, while in a non-approved state, all credit card transactions will be mocked (no actual money is used)._
    :::
</Step>

<Step 
  number="4" 
  title="Complete Game Details (Draft ‚Üí Review State)" 
  imagePath="/img/complete-game-details-infographic.png"
>
  1. Go to ***Games > List*** and click ***Edit(Complete)*** on your game
     - Complete the following required fields:
       - Cover Image URL (Minimum 800x600)
       - Game File (zip archive)
       - Add YouTube Gameplay Video ID
       - Price (in USD)
       - Game Mission / Objectives
  2. Perform your own review on the game. Run through all of these [checks](arcade-launch/game-submission-checklist.md)
  3. Once all the checks look good on your end. Set the game status to the _Review_ state
</Step>

  <Step 
  number="5" 
  title="Game Review Process (Review ‚Üí Approved/Rejected State)" 
  imagePath="/img/step-game-review-process.png"
  >
  - Our review team will evaluate your game (1 ~ 3 business days typical)
  - Status will update as follows:
    - _Approved_: You'll receive an email notification
    - _Rejected_: You'll receive an email with feedback and required changes

  ‚ö†Ô∏è If rejected, update your game and resubmit it by switching back to ***Review***.
  </Step>


  <Step 
  number="6" 
  title="Going Live (Approved ‚Üí Live State)" 
  imagePath="/img/go-live-step-6-infographic.png"
  >
  - Once approved, your game transitions to the Approved ‚Üí Live State
  - Your game will appear in the Luxodd Merchant Catalog
  - Arcade venue operators can now select and deploy your game
  - Your game becomes playable on real Luxodd arcade cabinets

  :::info
  _üöÄ The first time your game is selected by a merchant, it officially transitions to ***Live*** status._
  :::
  </Step>
